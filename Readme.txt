Squadfree - Free Bootstrap template for creative
Squadfree is a free HTML site template for creatives with simple and modern layout. You can use Squadfree to build all kind of multi purpose website like landing page, business landing page, marketing website, mobile app, software landing page, startup landing page and others.

Squadfree is created with powerful tools like Bootstrap framework, HTML5, CSS3. The code is well structured and organized, so, it is very easy to change and update the design. All elements are editable and customizable.

Feel free to customize and use this template for your next project.

Key Features:

Clean & Modern Design
Built with latest Bootstrap version
Sass/SCSS source files included (pro and membership users only)
Easy to customize, clean and commented code
PHP/Ajax contact form script (pro version only)
Fully customizable
One pager design
Smooth Scroll
Cross Browser compatible
Beautiful mobile menu
Responsive Layout
